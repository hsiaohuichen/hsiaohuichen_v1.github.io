Setting Goals and Objectives

What’s SBOX 1.4 about?
SBOX 1.4 deploys big data platforms like Splunk and Hadoop in minutes without subject matter expertise or DevOps. Once deployed, user can check the workflows for configuration and management of nodes, clusters, and infrastructure health on a single, simplified management screen. SBOX 1.4 also provides customized network configurations, and scheduled task and activities. Project Outcome Achievement - Improved overeall accessibility and usability - Less time to deploy the server indicates simplified process and successful navigation - Present the complicated storage UI in a simple and graphical way Creating Wireframes and User flows High-fidelity Prototypes Conducting

What are the goals of the project?
The goal of the interface is to help users to complete the server initiative through the simple process with the pleasant experience. The updates of user interface include wizard, homepage, and reboot improvement. Moreover, value management needs to be added in this version in order to let user to manage the storage of the server easier.
